# Fuentes Personalizadas

Coloca aquí los archivos de las fuentes personalizadas.

## Archivos Necesarios

### MoMo Trust Display (para textos)
- `MoMoTrustDisplay-Regular.woff2` (preferido)
- O `MoMoTrustDisplay-Regular.woff`
- O `MoMoTrustDisplay-Regular.ttf`

### Michroma (para títulos)
- `Michroma-Regular.woff2` (preferido)
- O `Michroma-Regular.woff`
- O `Michroma-Regular.ttf`

## Notas

- El sistema buscará automáticamente los archivos en este orden: `.woff2` → `.woff` → `.ttf`
- Solo necesitas tener uno de los formatos por fuente
- `.woff2` es el formato más moderno y eficiente (recomendado)
